# Shopify Order Dashboard App

A full-stack Shopify public app that displays merchant orders from the last 60 days with complete order details, line items, and images.

## Features

- ✅ OAuth 2.0 authentication with Shopify
- ✅ Fetch and display last 60 days' orders
- ✅ View detailed order information
- ✅ Line items with product images
- ✅ PostgreSQL database storage
- ✅ GraphQL API integration
- ✅ Responsive React dashboard
- ✅ Real-time order syncing

## Tech Stack

### Backend
- Node.js & Express.js
- PostgreSQL with pg driver
- Shopify GraphQL Admin API
- OAuth 2.0 authentication

### Frontend
- React 18 with Vite
- Tailwind CSS
- Axios for API calls
- Lucide React icons

## Prerequisites

- Node.js (v16 or higher)
- PostgreSQL (v12 or higher)
- Shopify Partner account
- ngrok or similar tunneling service (for development)
